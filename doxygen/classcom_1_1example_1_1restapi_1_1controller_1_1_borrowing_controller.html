Detailed Description

Controller class providing RESTful endpoints for borrowing and returning books.

This controller interacts with the BorrowingService and UserService classes, managing book borrowing actions and retrieval of borrowed book data.

The controller is hidden from Swagger documentation via the Hidden annotation.
